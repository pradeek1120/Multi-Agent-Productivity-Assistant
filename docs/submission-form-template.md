# Submission Form Template

Use this when filling the Hack2Skill submission form.

## Cloud Run Deployment Link

Paste your public Cloud Run service URL:

`https://YOUR_CLOUD_RUN_SERVICE_URL`

If you want to mention the demo endpoint in the PPT or video, use:

`https://YOUR_CLOUD_RUN_SERVICE_URL/demo/run`

## GitHub Repository Link

Paste your public GitHub repository URL:

`https://github.com/YOUR_USERNAME/YOUR_REPOSITORY`

## Demo Video Link

Paste a public video URL:

- YouTube unlisted/public link, or
- Google Drive public link

## Final Project PPT

Build the PPT using:

- [`docs/ppt-slides.md`](/home/devin/Multi-Agent-Productivity-Assistant/docs/ppt-slides.md)
- [`docs/submission-summary.md`](/home/devin/Multi-Agent-Productivity-Assistant/docs/submission-summary.md)
- [`docs/demo-script.md`](/home/devin/Multi-Agent-Productivity-Assistant/docs/demo-script.md)
- [`docs/architecture.md`](/home/devin/Multi-Agent-Productivity-Assistant/docs/architecture.md)

Export it as a PDF before upload.

## Final Check Before Submit

1. Cloud Run URL is public and working
2. GitHub repository is public
3. Demo video link is public
4. PPT is exported as PDF
5. Secrets are not committed to GitHub
